# Website Images

Place website image assets in this folder.

Current organization:

- Root level: Main brand assets and hero images
- `logo/`: Brand logos and icon files (planned)
- `hero/`: Homepage hero images (planned)
- `games/`: Game thumbnails and illustrations (planned)
- `team/`: Team member photos (planned)
- `about/`: About page visuals (planned)

Current files:

- `speelan-ai-logo.svg` — Main brand logo
- `auti_1.jpg` — Homepage hero background image

Example file names:

- `logo/logo.svg`
- `hero/home-hero.png`
- `games/routine-builder-thumbnail.png`
- `about/mission-illustration.png`

Tips:

- Use `public/images` for static image assets that should be served directly.
- Refer to images in Next.js with `/images/<file>` or `next/image`.
- Keep file names kebab-case and descriptive.
